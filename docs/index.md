<div class="center"><b>Astor Network</b> is an Ethereum testnet that uses SHA3 as its proo of work (PoW) method. SHA3 allows for a higher performance, and more decentralized proof-of-work Ethereum. This network is based on ECIP-1049 which is a proposal on Ethereum Classic created in response to the 51% attack of January 2019.</div>
